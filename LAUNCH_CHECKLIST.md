# 📋 Wedding Website Launch Checklist

Use this checklist to ensure everything is perfect before sharing with guests!

## Before Going Live

### Content ✓

- [ ] **Couple names updated** (config.json)
  - Bride name: ___________________
  - Groom name: ___________________

- [ ] **Couple photo added**
  - File name: `couple-photo.jpg`
  - Quality: High resolution (1000+ pixels)
  - Format: JPG or PNG

- [ ] **Wedding story written** (config.json)
  - Clear and romantic
  - 2-3 sentences
  - No typos

- [ ] **Wedding date correct** (config.json)
  - Date: ___________________
  - Day: ___________________

- [ ] **Wedding time correct** (config.json)
  - Time: ___________________
  - ⚠️ Double-check timezone!

- [ ] **Venue details accurate** (config.json)
  - Venue name: ___________________
  - Full address: ___________________
  - Has street address & city

- [ ] **Events section filled** (config.json)
  - Ceremony time: ___________________
  - Reception time: ___________________
  - Other events: ___________________

- [ ] **Dress code is clear** (config.json)
  - Description: ___________________

### Guest List ✓

- [ ] **All 300 guests added** (guests.json)
  - Total count: _____ guests
  - Names are spelled correctly
  - Names are sorted (optional but recommended)
  - No duplicates
  - No empty entries

- [ ] **Names can be found easily**
  - Sorted alphabetically
  - Common name variations included (e.g., "Rob" AND "Robert")
  - Special characters handled (e.g., names with accents)

### Deployment ✓

- [ ] **Deployed to Netlify** (or Vercel/GitHub Pages)
  - Deployment service: ___________________
  - Website URL: ___________________

- [ ] **Website loads correctly**
  - All photos displaying
  - No broken styling
  - All text visible
  - No error messages in console (F12)

- [ ] **RSVP form works**
  - Can select name from dropdown
  - Can choose attendance option
  - Can select plus-ones
  - Can select dietary preference
  - Can add special requests
  - Can submit form
  - Gets success message

### Mobile Testing ✓

- [ ] **Tested on iPhone**
  - Portrait & landscape modes
  - All text readable
  - Buttons work
  - Form submits

- [ ] **Tested on Android**
  - Portrait & landscape modes
  - All text readable
  - Buttons work
  - Form submits

- [ ] **Tested on tablet**
  - Layout looks good
  - No horizontal scrolling
  - Form is user-friendly

### WhatsApp Testing ✓

- [ ] **Link works on WhatsApp**
  - Send test message in WhatsApp
  - Click link from WhatsApp
  - Website loads correctly
  - Website preview looks good

- [ ] **Link works on desktop browser**
  - Copy and paste URL
  - Works in Chrome, Firefox, Safari

### Optional Features ✓

- [ ] **Email notifications** (if using Formspree)
  - [ ] Formspree account created
  - [ ] Form ID added to config.json
  - [ ] Test RSVP submitted
  - [ ] Email received successfully

- [ ] **Custom domain** (optional)
  - [ ] Domain purchased (if wanted)
  - [ ] Connected to Netlify/Vercel
  - [ ] URLs updated

## Before Sending Invites

### Security & Privacy ✓

- [ ] **No sensitive information visible**
  - No bank details
  - No personal phone numbers in story
  - No private email addresses visible

- [ ] **Privacy settings correct**
  - Each guest sees only their page
  - No guest list visible to others
  - RSVPs are private

### Info Double-Check ✓

- [ ] **Verify with couple**
  - Names spelled correctly
  - Photo approved
  - Story approved
  - Date confirmed
  - Time confirmed
  - Venue confirmed
  - Dress code approved

- [ ] **Spell check everything**
  - Names
  - Story
  - Venue address
  - All text fields

### Guest List Final Check ✓

- [ ] **Guest count verified**
  - Total guests: _____ (should be ~300)
  - No duplicates counted
  - No test names left in list

- [ ] **Common names included**
  - For people with multiple names
  - Maiden names if needed
  - Preferred names

- [ ] **Organizational harmony**
  - Asked couple for final guest list
  - Confirmed no one is missing
  - Confirmed no extras added
